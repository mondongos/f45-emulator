# F45 Emulator
Emulating the software used by F45

## Server
### Database Tables 
- `workouts`
    - creator_id 
    - obj: exercises
    - category 
- `exercises`
    - clip (gif) link
    - workout_id
    - general info
- `creators` (LATER)
    - general info (i.e. name, age, etc)
